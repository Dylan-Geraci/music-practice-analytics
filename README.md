## Why SessionLog?

Most musicians practice without data. SessionLog gives you the insights you need to practice smarter, stay consistent, and see real progress.

**Built for musicians who want to:**
- Track every practice session with detailed metrics
- Visualize practice patterns and identify gaps
- Set and achieve weekly practice goals
- Build long-term consistency through streak tracking
- Share progress with teachers or the community

---

## Core Features

### Practice Tracking
Log sessions with duration, tempo, accuracy ratings, and notes. Track what you practiced, when you practiced, and how it went.

### Analytics & Insights
Visualize your practice activity with heatmaps, charts, and trends. See which songs get the most attention and where you're improving.

### Goal Management
Set daily or weekly practice targets. Monitor progress and stay accountable to your goals.

### Song Library
Organize your repertoire by song and section. Track practice history and progress for each piece you're learning.

### Streak Building
Build consistency with streak tracking. See your longest streaks and current momentum.

### Public Profiles
Share your practice stats publicly or keep them private. Perfect for teachers, students, or anyone building in public.

## Built With

SessionLog is built on modern, production-grade infrastructure:

- **Next.js 15** - Fast, server-rendered React application
- **Supabase** - Secure PostgreSQL database with real-time capabilities
- **Vercel** - Global edge deployment for low latency
- **TypeScript** - Type-safe codebase for reliability

---

## Security & Privacy

- End-to-end encryption for sensitive data
- Row-level security on all database operations
- Email verification for new accounts
- Configurable privacy controls for public profiles
- SOC 2 compliant infrastructure (Supabase + Vercel)
